# DSA

This repository contains my Data Structures and Algorithms (DSA) practice solutions and revision notes.

For each problem, I may include:

* Brute Force Approach
* Better Approach
* Optimal Approach
* Pattern Used
* Key Insights
* Time and Space Complexity

## Purpose

* Track my DSA learning journey
* Store solutions for problems from LeetCode, GFG, Striver's Sheet, contests, and other platforms
* Maintain notes for quick revision
* Preserve solutions that may be difficult to find again later

The focus is on understanding patterns and problem-solving rather than memorizing solutions.
